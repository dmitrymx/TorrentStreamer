## [1.0.0] — 2026-05-16

### 🎉 Первый релиз

#### Добавлено
- **HTTP-стриминг** — воспроизведение музыки прямо во время скачивания через локальный HTTP-сервер (`client.createServer()`)
- **Встроенный аудио-плеер** — Play/Pause, Prev/Next, Seek, Volume, автопереход
- **Двойной клик по треку** — мгновенное воспроизведение любого файла из списка (встроенный и внешний плеер)
- **Поддержка внешних плееров** — foobar2000, AIMP, VLC, Winamp, MPC-HC с оптимальными CLI-аргументами
- **HTTP stream URLs для внешних плееров** — решает проблему блокировки файлов (WebTorrent lock)
- **Автоопределение плееров** — сканирование Program Files на наличие поддерживаемых плееров
- **Обложки альбомов** — автоматический поиск cover.jpg/folder.jpg/front.png и отображение через Data URL
- **Поддержка магнет-ссылок и .torrent файлов** — санитизация магнет-ссылок (удаление \\r\\n, пустых трекеров)
- **Режим «Только прослушивание»** — загрузка во временную папку с автоочисткой
- **Сворачивание в трей** — иконка в системном трее с контекстным меню
- **Тёмная и светлая тема** — переключение в настройках
- **Прогресс файлов в реальном времени** — обновление каждые 500мс
- **Статистика загрузки** — скорость, пиры, ETA, размер
- **14 публичных трекеров** — opentrackr, openbittorrent, exodus, rutracker и др.
- **Кнопка отмены подключения** — прерывание загрузки метаданных
- **Кнопка очистки поля ввода** — быстрая очистка магнет-ссылки
- **Portable .exe** — один файл, не требует установки

#### Технические детали
- Electron 34 + WebTorrent 2.5
- `file.streamURL` для корректных HTTP URL (Windows-совместимость путей)
- Graceful shutdown: HTTP-сервер → WebTorrent client → temp cleanup
- contextBridge для безопасного IPC
- Sequential download strategy для оптимального стриминга
